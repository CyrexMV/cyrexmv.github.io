Was ist ein Kondensator?

Ein Kondensator ist ein passives elektrisches Bauelement, das elektrische Energie in einem elektrischen Feld speichert. Man kann ihn sich wie einen kleinen, extrem schnell aufladbaren Akku vorstellen.

Merke: Kondensatoren speichern keine Ladung dauerhaft, sondern geben sie bei Bedarf blitzschnell wieder ab!

Anwendungsbereiche

  • Glättung/Siebung: In Netzteilen wandeln sie pulsierenden Gleichstrom in eine saubere, stabile Gleichspannung um.
  • Zeitglieder: Zusammen mit einem Widerstand können sie Lade- und Entladevorgänge für Timer und Taktgeber steuern.
  • Filter: Sie blockieren Gleichstromanteile und lassen Wechselstrom passieren (Koppelkondensator) oder filtern Frequenzen.

Beschriftung und Farbcodes

Die Werte von Kondensatoren werden oft durch einen Code angegeben. Die Basiseinheit für diese Codes ist fast immer Pikofarad (pF).

Gedruckte Zahlencodes (Beispiel "104")

Auf vielen Keramikkondensatoren findet man einen dreistelligen Code wie 104.

  • 1. Ziffer: Erster Wert (hier: 1)
  • 2. Ziffer: Zweiter Wert (hier: 0)
  • 3. Ziffer: Multiplikator (Anzahl der Nullen, hier: 4)

Die Berechnung ist also: 10 + "0000" pF = 100.000 pF = 100 nF = 0,1 µF.
